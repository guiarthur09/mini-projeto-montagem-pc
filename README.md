# 🖥️ Projeto: Montagem de PC

## 📌 Descrição
Este projeto consiste na simulação e documentação da montagem de um computador, abordando desde a escolha dos componentes até a montagem final.

O objetivo é demonstrar conhecimentos básicos de hardware, compatibilidade de peças e organização de um setup funcional. Também serve como um guia introdutório para quem deseja aprender a montar um PC.

---

## 🎯 Objetivos
- Entender os principais componentes de um computador
- Aprender sobre compatibilidade entre peças
- Simular o processo de montagem de um PC
- Desenvolver organização e planejamento de hardware

---

## 🛠️ Componentes Utilizados
- Processador (CPU)
- Placa-mãe
- Memória RAM
- Armazenamento (HD/SSD)
- Fonte de alimentação
- Gabinete
- Placa de vídeo (opcional)

---

## 💡 Considerações
Durante o projeto, foram levados em conta fatores como:
- Custo-benefício
- Desempenho
- Finalidade do computador (uso diário, estudos ou jogos)

---

## 🚀 Possíveis Melhorias
- Comparação entre diferentes configurações
- Inclusão de imagens ou diagramas da montagem
- Testes de desempenho (benchmark)

---

## 📚 Aprendizados
Este projeto ajudou a desenvolver conhecimentos sobre hardware, montagem de computadores e planejamento de sistemas, sendo uma base importante para quem deseja seguir na área de tecnologia. Além disso nos ajudando a entender melhor o trabalho em trabalhar em equipe.
