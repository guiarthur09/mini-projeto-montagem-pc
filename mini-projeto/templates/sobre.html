💭 Sobre o Projeto

Este projeto foi desenvolvido como uma solução prática para o gerenciamento e montagem de computadores, permitindo o cadastro de configurações personalizadas e a visualização técnica de cada máquina.

Utilizamos tecnologias de desenvolvimento web para criar uma interface intuitiva, focada na experiência do usuário e na organização de dados de hardware.

🧑‍💻Equipe

O desenvolvimento deste sistema é fruto de estudos em engenharia de software e análise de sistemas, visando a automação de processos de suporte técnico.
